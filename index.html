How to Play

  • Tap a tube to select the top ball
  • Tap another tube to move the ball there
  • You can only move a ball to an empty tube or on top of the same color
  • Complete the level when each tube contains balls of only one color
  • Empty tubes can be used for temporary storage

Try to solve each level in the minimum number of moves!
